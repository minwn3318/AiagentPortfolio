
# 도서 관리 시스템 bookmuseum (백엔드)
> 제작시간 : 2025-10-28 ~ 2025-11-03

## 1. 프로젝트 소개
취업을 위해 면접준비를 하지만 면접 준비 시 피드백을 제때제때 확인하거 받을 수 없어 불편합니다.   
본 프로젝트는 1인 면접 준비 프로젝트로 질문자에게 이력서를 받고 이를 바탕으로 질문을 한 후 최종 평가를 내려 적절한 피드백을 주는 AI Agent 프로젝트 입니다

## 2. 주요 기능
- 이력서 분석
- 질문 생성
- 질문 분류 및 평가
- 분류별 세부 질문 생성
- 최종 평가
   
## 3. 기술 스택
- RAG Chain
- Lang Graph
- Lang Pipeline
- Vector DB

## 4. 빌드 및 실행
